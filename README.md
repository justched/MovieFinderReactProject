# 🎬 Movie Finder

A lightweight, responsive React application that allows users to search for TV shows and movies in real-time using the **TVMaze API**.

## ✨ Features

* **Instant Search:** Fetch show details, ratings, and posters instantly.
* **Smart Layout:** The UI transitions from a centered landing view to a top-aligned results grid upon searching.
* **Error Handling:** Built-in alerts for empty searches or API fetch failures.
* **Fallback Assets:** Automatically displays a placeholder image if a show poster is unavailable.
* **User Friendly:** Supports "Enter" key functionality for quick searching.

## 🛠️ Tech Stack

* **Library:** React.js
* **Styling:** CSS3 (Custom Styles)
* **API:** [TVMaze API](https://www.tvmaze.com/api)
* **Icons:** Emoji-based UI
